Tasmotizer Gang은 [Tasmotaizer](https://github.com/tasmota/tasmotizer) 를 기반으로 합니다.<br/>
멀티 upload 및 멀티 send config 등을 지원하며, macOS 와 윈도우 환경에서 사용 가능합니다.

(현재는 ESP8266만 지원합니다)
 
## 설치 및 실행

- Option 1: pip를 사용하여 PyPI를 통해 설치<br/>
pip와 wheel packages를 업그레이드 합니다 : `pip3 install --upgrade pip wheel`<br/>
Tasmotizer Gang을 설치합니다 : `pip3 install tasmotizer`<br/>
마지막으로 쉘/CLI에서 `python3 -m tasmotizer.py` 를 통해 Tasmotizer Gang을 실행합니다.

- Option 2: Release된 실행파일 사용<br/>
우측 Releases에서 최신 Tasmotizer Gang 실행 파일을 설치하고 이를 실행합니다.
